# Pathfinder Demo (PWA)

This is a **front-end first** prototype for EWOR.
- localStorage persistence
- demo AI outputs (deterministic templates)
- installable as a PWA (basic manifest + service worker)

## Run
```bash
npm install
npm run dev
```

## Pages
- /today — daily pulse
- /decisions — decision journal + demo guidance
- /weekly — weekly opportunity plan
- /habits — habits + affirmations
